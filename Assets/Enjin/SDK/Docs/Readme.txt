To view the documentation. Unzip the SDKDocs into a folder. Once the files have been unzipped, go into the
folder that contains the docs.html file. Double click on this file to load them into a local browser session.

Steps:
- Unzip SDKDocs.zip file
- Navigate to folder where docs.html is
- Double-click on docs.html file to start a local browser session of the docs.

Future documentation will be hosted online through a portal in which developers can access on the web.
